Mjukvara

Mjukvaran till Anoroc kan delas upp i fyra kategorier.

  • Frontend, hemsidan, byggd med HTML, CSS & JavaScript
  • GUI, grafiska användssnittet som används för att styra Anoroc
  • Fordonskontroll, backend som GUI:n använder för att styra Anoroc
  • Firmware, mjukvaran som körs på Anoroc

All kod finns på github för den som är intresserad.
